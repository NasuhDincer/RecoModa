Project Description

Online Language Learning Platform

The aim of this project is to implement an online language learning platform that satisfies all the requirements for teachers, students, language natives, and admin. The system will allow students to request classes according to their levels and create requests for online meetings with natives for speaking exercises. The teachers can give lessons, assign homework to students which they choose, and grade the exams and homework. Additionally, the teachers can see how many students are taking their class and track the students' activities on student accounts. The natives can give speaking exercises and grade them. A native can be a teacher. Lastly, the admin can see the statistics about courses, teachers, and lessons.
